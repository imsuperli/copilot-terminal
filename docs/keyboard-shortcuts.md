# 键盘快捷键清单

## 全局快捷键（终端视图）

这些快捷键在终端视图中全局生效：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Tab` | 打开快速切换面板 | 模糊搜索窗口名称或路径，支持键盘导航 |
| `Ctrl+B` | 切换侧边栏展开/收起 | 侧边栏显示窗口列表 |
| `Ctrl+1~9` | 切换到第 N 个窗口 | 按侧边栏顺序切换（1=第一个，2=第二个...） |
| `Escape` | 关闭打开的面板 | 关闭快速切换面板，如果没有面板打开则传递给终端 |

## 终端内快捷键

这些快捷键在终端输入区域生效：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+C` | 复制选中文本 | 如果有选中文本则复制，否则发送中断信号 |
| `Ctrl+V` | 粘贴剪贴板内容 | 本地终端直接粘贴文本；SSH 终端中若剪贴板是图片，则优先上传图片并提示远端路径 |
| `Ctrl+Enter` | 插入换行符 | 用于多行输入（如 Claude Code、Python REPL） |
| `Shift+Enter` | 插入换行符 | 同 Ctrl+Enter，由 xterm.js 原生处理 |

## 已释放的快捷键

| 快捷键 | 原功能 | 释放原因 |
|--------|--------|----------|
| `Ctrl+P` | 打开快速切换面板 | 用户要求释放，改用 Ctrl+Tab |
| `Ctrl+Shift+Tab` | 向后切换窗口（MRU 顺序） | TabSwitcher 功能已移除 |

## 快捷键优先级

1. **应用级快捷键优先**：`Ctrl+Tab`、`Ctrl+B`、`Ctrl+1~9` 会被应用拦截，不会传递到终端
2. **终端内特殊处理**：`Ctrl+C`、`Ctrl+V`、`Ctrl+Enter` 在终端内有特殊行为
3. **SSH 图片粘贴**：仅在 SSH pane 且剪贴板内容可识别为图片时触发上传；非图片和空剪贴板仍按普通文本粘贴处理
3. **其他按键透传**：所有其他按键（包括现在的 `Ctrl+P`）都会正常传递给终端 shell

## 注意事项

- 快捷键只在对应窗口激活时生效（`enabled: isActive`）
- 在 xterm.js 的 textarea 中，只有明确定义的快捷键会被拦截，其他按键正常传递
- `Escape` 键的行为是智能的：有面板打开时关闭面板，否则传递给终端（如 vim 的退出）
