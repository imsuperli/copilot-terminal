# 快捷键修改说明

## 修改内容

将快捷导航面板的唤出快捷键从 `Ctrl+K` 改为 **双击 Shift 键**。

## 实现原理

1. 监听全局 `keydown` 事件
2. 检测 `Shift` 键按下
3. 记录每次按下 Shift 的时间戳
4. 如果两次按下的时间间隔小于 300ms，则触发快捷导航面板
5. 触发后重置时间戳，避免连续触发

## 技术细节

- 使用 `useRef` 存储上次按下 Shift 的时间戳
- 时间间隔阈值：300ms（可调整）
- 支持切换面板状态（打开/关闭）

## 使用方法

在应用的任何界面（主页或终端视图）快速双击 `Shift` 键即可唤出快捷导航面板。

## 优势

- 更符合现代应用的交互习惯（类似 IntelliJ IDEA 的 Shift+Shift 搜索）
- 单手操作更方便
- 不会与其他快捷键冲突
