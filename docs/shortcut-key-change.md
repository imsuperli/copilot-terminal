# 快捷键修改说明

## 修改内容

将快捷导航面板的默认唤出快捷键改为 **双击 Ctrl 键**。

## 实现原理

1. 监听全局 `keydown` 事件
2. 检测配置的双击按键
3. 记录每次按键松开的时间戳
4. 如果两次完整按键的松开时间间隔小于阈值，则触发快捷导航面板
5. 触发后重置时间戳，避免连续触发

## 技术细节

- 使用 `useRef` 存储上次配置按键松开的时间戳
- 时间间隔阈值：400ms（可调整）
- 支持切换面板状态（打开/关闭）

## 使用方法

在应用的任何界面（主页或终端视图）快速双击 `Ctrl` 键即可唤出快捷导航面板。

## 优势

- 单手操作更方便
- 不会与其他快捷键冲突
